# ひとりごとメモ - ホーム画面に追加する手順

## できあがるもの
このフォルダの4つのファイル(index.html, manifest.json, sw.js, icon)を
インターネット上のどこかに置くと、スマホの「ホーム画面に追加」で
アイコンタップだけで開ける、ほぼアプリのようなメモ帳になります。

データはスマホのブラウザ内(localStorage)に保存されます。
他の端末とは同期しません。

---

## 手順1: GitHubアカウントを作る(持っていなければ)
https://github.com/ にアクセスし、無料で登録します。

## 手順2: 新しいリポジトリを作る
1. 右上の「+」→「New repository」
2. Repository name に好きな名前(例: chat-memo)を入力
3. Public を選択
4. 「Create repository」をクリック

## 手順3: ファイルをアップロードする
1. 作ったリポジトリのページで「Add file」→「Upload files」をクリック
2. このフォルダの中身(index.html, manifest.json, sw.js, icon-192.png, icon-512.png)
   をまとめてドラッグ&ドロップ
3. 下にスクロールして「Commit changes」をクリック

## 手順4: GitHub Pagesを有効にする
1. リポジトリの「Settings」タブを開く
2. 左メニューの「Pages」をクリック
3. 「Build and deployment」→「Branch」を `main` / `/ (root)` に設定して保存
4. 数十秒〜数分待つと、ページ上部に
   `https://(あなたのユーザー名).github.io/(リポジトリ名)/`
   というURLが表示されます

## 手順5: スマホでホーム画面に追加
1. 手順4で表示されたURLをスマホのブラウザ(SafariまたはChrome)で開く
2. 共有ボタン(Safari)またはメニュー(Chrome)から
   「ホーム画面に追加」を選択
3. ホーム画面にアイコンが追加されます。タップするとアプリのように起動します

---

## 更新したいとき
コードを直したら、同じ手順3の要領で index.html を上書きアップロードすれば
ホーム画面のアプリにも反映されます(ブラウザがキャッシュしている場合は
一度アプリを完全に閉じて開き直すと反映されやすいです)。

## うまく動かないとき
- アイコンが表示されない → manifest.json と icon-192.png/icon-512.png が
  同じフォルダにアップロードされているか確認してください
- 真っ白な画面になる → index.html 単体ではなく、フォルダごと
  (manifest.json, sw.js も含めて)アップロードされているか確認してください
